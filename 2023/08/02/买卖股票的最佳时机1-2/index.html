买卖股票的最佳时机1~2

动态规划 贪心
算法

阅读时长:   4 分

一、题目描述
给定一个数组 prices ，它的第 i 个元素 prices[i] 表示一支给定股票第 i 天的价格。

你只能选择 某一天 买入这只股票，并选择在 未来的某一个不同的日子 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 0 。

假设给定一个数组[7,1,5,3,6,4]

描绘的图表如下：

数组中每个值代表一个点，从图中我们很容易得出，最大利润就是图中的两个峰值差，也就是说，我们只要维护数组中的最小值，再找到最大值就能够得出最大利润了。

我们发现，股票要先买入才能卖出，所以对于数组prices,如果prices[i]<prices[i-1],我们就可以买入，否则就卖出，这样才能实现盈利而不会亏损。

public int maxProfit(int[] prices) {
      int n = prices.length;
      int max = 0;
      int dp =prices[0];//可以买入股票的最小值
      for(int i=1;i<n;i++) {
          if(prices[i]<prices[i-1]) {
              //价格低于前一天，可以买入
              dp = Math.min(prices[i],dp);
          } else {
              //价格上涨，可以卖出
              max = Math.max(prices[i] - dp,max);
          }
      }
      return max; 


  }

二、题目描述：

给你一个整数数组 prices ，其中 prices[i] 表示某支股票第 i 天的价格。

在每一天，你可以决定是否购买和/或出售股票。你在任何时候 最多 只能持有 一股 股票。你也可以先购买，然后在 同一天 出售。

返回 你能获得的 最大 利润 。

示例 1：

输入：prices = [7,1,5,3,6,4]
输出：7
解释：在第 2 天（股票价格 = 1）的时候买入，在第 3 天（股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5 - 1 = 4 。
     随后，在第 4 天（股票价格 = 3）的时候买入，在第 5 天（股票价格 = 6）的时候卖出, 这笔交易所能获得利润 = 6 - 3 = 3 。
     总利润为 4 + 3 = 7 。

我们发现题二在题一的基础上做了一些改变，
股票可以多次买入卖出。

当然，为了获取最大利润，我们还是要低价买入高价卖出，

我们可以在题一的基础上做些变动，因为可以多次买入，我们可以在卖出后再买入当前股票，同时维护之后买入股票的最小值，以求获取最大利润。

public int maxProfit(int[] prices) {
        int n = prices.length;
        int max = 0,sum_max = 0;
        int dp =prices[0];//可以买入股票的最小值
        for(int i=1;i<n;i++) {
            if(prices[i]<prices[i-1]) {
                //价格低于前一天，可以买入
                dp = Math.min(prices[i],dp);
            } else {
                //价格上涨，可以卖出
                max = Math.max(prices[i] - dp,max);
                //卖出后，可以考虑买入新的股
                dp = prices[i];
                sum_max+=max;
                max = 0;//更新max值，找下一个可以获得的最大利润
            }
        }
        return sum_max; 


    }

再上面代码中我们发现如果prices[i]>prices[i-1]时，都会更新卖出的最大利润
max = Math.max(prices[i] - dp,max);也就是说，如果股票价格比前一天有所上涨，我们可以持续卖出再买入。否则就可以只买入不卖出。这样我们可以得到另一种解法贪心。

public int maxProfit(int[] prices) {
      int n = prices.length;
      int sum_max = 0; // 用于记录多次交易的最大利润之和
      for (int i = 1; i < n; i++) {
          // 如果当前价格大于前一天的价格，表示可以在前一天卖出并在当天买入，将利润累加到sum_max中
          if (prices[i] > prices[i - 1]) {
              sum_max += prices[i] - prices[i - 1];
          }
          // 如果当前价格小于等于前一天的价格，则不进行买卖操作
      }

      return sum_max;
  }

动态规划其实就是一种状态转移过程，把状态转移方程写出来了，具体实现也就不难了。
